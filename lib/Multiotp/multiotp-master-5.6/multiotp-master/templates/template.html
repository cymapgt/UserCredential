PERSONAL AND CONFIDENTIAL PERSONNEL ET CONFIDENTIEL / {MultiotpUserDescriptionUC}

To authenticate yourself on your company network, you will have to use a strong authentication password also known as an OTP code. This code will be generated by your smartphone and will change every time you use it.

The generated code is valid for the following account: {MultiotpUserAccount}
and you will have to type your network password as a prefix of the displayed code.
and you will have to use the following prefix PIN: {MultiotpUserPin}
And on your code generator, you will have to use the following PIN: {MultiotpUserPin}

Serial number of the attached physical code generator device (token): {MultiotpUserTokenSerial}

In order to install the code generator on your smartphone, you will have to follow the next three steps.

 

1) QRcode reader application installation

If your smartphone does not have a QRcode app yet, please install now such an application (like "ZXing barcode scanner" for Android, "Quick Scan" for iOS and "i-nigma" for Windows Phone).

 

2) OTP generator application installation

Install a compatible application on your smartphone. You can easily install this application by scanning (or clicking) this QRcode (valid for Android, iOS, Windows Phone, BlackBerry):

{MultiotpQrCodeUrl url=http://go.6co.ch/otpauth width=200 height=200}
FreeOTP Authenticator / Authenticator / 2 Steps Authenticator
{MultiotpQrCodeUrl url=http://go.6co.ch/motp-xml width=200 height=200}
OTP Authenticator / Yamotp / OTP Authenticator for Java ME

 

3) Token provisioning

The last step consists in inserting your personal token key generator into the application. Simply scan the QRcode printed below (or click on it directly from your smartphone).

{MultiotpQrCodeUserToken width=250 height=250}
{MultiotpUserDescription}

If you need to generate your token with another application that doesn't support QRcode provisioning, you have all the details needed below:

  • algorithm: {MultiotpUserAlgorithm}
  • secret seed: {MultiotpUserTokenSeed}
       (or {MultiotpUserTokenSeedBase32} in Base32, like for Google Authenticator)
  • number of digits: {MultiotpUserTokenNumberOfDigits}
  • time interval: {MultiotpUserTokenTimeInterval}
  • next event: {MultiotpUserTokenNextEvent}

2) Installation de l'application de génération des codes OTP à usage unique

Il faut maintenant installer une application compatible sur votre smartphone. L'installation se fait très simplement en scannant (ou en cliquant) sur le QRcode ci-dessous (valable pour Android, iOS, Windows Phone, BlackBerry):

3) Configuration de votre générateur personnel
